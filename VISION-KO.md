## 철학적 배경: 장회익 교수님의 자연철학

장회익 교수님은 이미 한국에서 그 틀을 만들어 놓으셨습니다.

### 《자연철학강의》의 구조 - 심우십도(十牛圖)

| 단계 | 심우십도 | 자연철학 대응 | 우리의 학습 |
|------|----------|---------------|-------------|
| 1장 | 소를 찾아 나서다 | 앎의 바탕 구도 | SICP (계산적 사고) |
| 2장 | 소의 자취를 보다 | 고전역학 | SICM |
| 3장 | 소를 보다 | 상대성이론 | FDG → 일반상대성 |
| 4장 | 소를 얻다 | 양자역학 | Emmy + 양자역학 |
| ... | ... | ... | ... |
| 10장 | 저자거리로 돌아오다 | 온생명, 앎과 존재의 통합 | 보편학 |

### 핵심 통찰

> "칸트는 단순한 사상가가 아니라 과학자였다.
> 과학의 존립 근거를 탐구한 메타 과학자였다."
> — 장회익, 《물질, 생명, 인간》

- *칸트 철학과 현대 물리학*의 연결
- *존재와 앎*, 그것의 조화를 통한 온생명의 추구
- 의식 주체로서의 온생명

> "비유를 통해 설명하거나, 일화를 나열하기보다는

## 왜 SICM/FDG/Emmy인가?

### 표기와 지식의 통합

기존 교육의 문제:
- 비유로만 설명 → 정확한 이해 불가
- 손으로 수식 → 일반인에게 부담
- 전통 표기법 → 모호성, 기계가 해석 불가

SICM/Emmy의 해결:
- *수식 = 코드*: 함수형 표기법으로 정확성 확보
- *즉각적 피드백*: 이해를 코드로 검증
- *재현 가능한 앎*: 에이전트도 참여 가능

### 에이전트가 협력하는 학습

```
전통적 학습:
  교과서 → 손으로 풀이 → (혼자 고민) → 불확실한 이해

에이전트 협력 학습:
  책 (org-mode) → 코드 실행 → 에이전트 질문/설명 → 검증된 이해
                     ↑                    ↓
                  Emmy/Scheme ←─────── 코드로 표현
```

## 궁극적 목표

> "그게 완성이 되면 아이들에게 현대 과학지식의 전체 틀을
> 명확하게 전달할 수 있을 것이다.
> 여기에 에이전트가 확실히 협력할 수 있는 체계가 완성되는 것이다."

### 보편학 (Universal Science)의 기반

- Seymour Papert의 Logo → Gerald Sussman의 SICM
- 수학적 도구로서 *수식과 코드가 동일*
- 인간지능과 인공지능이 *표기법을 공유*

### 학습 체계

1. SICP: 계산적 사고의 기초
2. SICM: 고전역학 (라그랑주, 해밀턴)
3. FDG: 미분기하학
4. 양자역학, 상대성이론으로 확장
5. 온생명, 자연철학으로 통합
